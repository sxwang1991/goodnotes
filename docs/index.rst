=============================
scikit-fem文档
=============================

`scikit-fem <https://github.com/kinnala/scikit-fem>`_ 是一个轻量的 
实施 `有限元装配 <https://en.wikipedia.org/wiki/Finite_element_method>`_ Python 3.7+库. 
其主要目的就是将双线性形式转化成为稀疏矩阵、将线性形式转化成为向量.  
该库支持三角形网格、四边形网格、四面体网格、六面体网格、一维问题.

.. note::

    安装该库只需运行
    
    .. code-block:: bash

        pip3 install scikit-fem

    在 `图库 <listofexamples.html>`_ 中可以找到一些例子.
